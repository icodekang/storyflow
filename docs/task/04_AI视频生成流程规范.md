# AI 视频生成流程规范（pi-mono Agent Pipeline）

> 本文档定义 7 个 Agent 的详细输入输出规范、数据格式、以及 pi-mono Core 在各环节的使用方式。
> pi-mono 是每个 Agent 的运行时底层：Session 管理、三层记忆、工具调用全部由 pi-mono Core 提供。

---

## 通用规则

### pi-mono 执行上下文

每个 Agent 的 `execute()` 调用都发生在 pi-mono Core 的上下文中：

```
Agent.execute(context: AgentContext)
    │
    ├─→ SessionManager.get_session(session_id)  ← 获取当前 Session
    ├─→ MemoryLayer.search(query)                ← 检索长期记忆
    ├─→ _build_prompt(context, memory_results)   ← 构建 prompt（含记忆注入）
    ├─→ LLM API (via Tools Registry)            ← 调用 LLM
    ├─→ _parse_output()                        ← 解析输出
    └─→ MemoryLayer.add(agent_name, output)    ← 写入记忆
```

### 数据传递规范

- 每个 Agent 输出 JSON 格式的 `dict`，作为下一个 Agent 的 `input_data` 的一部分
- 每个 Agent 输出必须包含：`{ "agent_name": "...", "timestamp": "ISO8601", "data": {...} }`
- 数据 key 不与输入 key 冲突（除非明确覆盖）

### pi-mono Memory 使用规范

**执行前**：`memory.search(session_id, query, top_k=3)`
- query 使用当前 Agent 输入中的 `story_summary` 或 `description`
- 跨 story 检索：`session_id` 相同时可跨 story 检索同类型情节/角色

**执行后**：`memory.add(session_id, story_id, agent_name, output)`
- 中期记忆：写入 Session JSON 文件
- 长期记忆：向量入库（ChromaDB）

### 人工干预注入

`human_feedback` 非空时：
- prompt 末尾追加：`"\n\n【人工修改建议】：{human_feedback}\n请根据此建议重新生成输出。"`
- Agent 保留原输出有效部分，只调整 feedback 指出问题的地方
- 重新执行后，记忆记录更新为新版本

## Agent 详细规范

### Agent 1：ScriptAnalysis（剧本分析）

**文件**：`app/agents/script_analysis.py`

**输入**：`{ "story_text": "string" }`

**输出**：
```json
{
  "agent_name": "ScriptAnalysis",
  "timestamp": "ISO8601",
  "data": {
    "story_summary": "string（200字以内）",
    "genre": "string（剧情/科幻/奇幻/爱情/悬疑/惊悚/动画等）",
    "sub_genre": "string",
    "tone": "string（整体基调）",
    "target_audience": "string",
    "themes": ["string（核心主题列表）"],
    "mood_tags": ["string（氛围标签，用于视觉生成）"],
    "estimated_duration_sec": "number",
    "language": "string（默认 zh-CN）"
  }
}
```

**记忆检索 query**：`story_text` 全文

**Mock → Real**：
- Mock：正则 + 规则匹配
- Real：调用 LLM（GPT-4o / Claude）做摘要和分类

---

### Agent 2：PlotDeconstruct（情节拆解）

**文件**：`app/agents/plot_deconstruct.py`

**输入**：
```json
{
  "story_text": "string",
  "story_summary": "string（来自 Agent1）",
  "genre": "string",
  "tone": "string"
}
```

**输出**：
```json
{
  "agent_name": "PlotDeconstruct",
  "timestamp": "ISO8601",
  "data": {
    "act_structure": {
      "act1": { "title": "string", "description": "string", "ending_hook": "string" },
      "act2": { "title": "string", "description": "string", "rising_action": [], "midpoint_twist": "string" },
      "act3": { "title": "string", "description": "string", "climax": "string", "resolution": "string" }
    },
    "key_plot_points": [
      { "point_id": "number", "type": "string", "description": "string", "emotion": "string" }
    ],
    "conflict": "string",
    "pacing": "string"
  }
}
```

**记忆检索 query**：`story_summary`

---

**输出**：
```json
{
  "agent_name": "ScenePlanning",
  "timestamp": "ISO8601",
  "data": {
    "scenes": [
      {
        "scene_id": "string（sc_01...）",
        "scene_number": "number",
        "act": "number",
        "description": "string（30-80字）",
        "location": "string",
        "time_of_day": "string（day/night/dusk/dawn/morning）",
        "weather": "string（可选）",
        "emotion": "string",
        "camera_angle": "string（wide/medium/close-up/drone）",
        "transition": "string（cut/fade/dissolve/s_wipe）",
        "duration_sec": "number",
        "depends_on_scenes": ["string（可选）"]
      }
    ],
    "total_scenes": "number",
    "estimated_total_duration": "number"
  }
}
```

  "timestamp": "ISO8601",
  "data": {
    "characters": [
      {
        "character_id": "string（char_01...）",
        "name": "string",
        "role": "string（protagonist/antagonist/supporting）",

          "distinctive_features": ["string"]
        },
        "personality": "string",
        "backstory": "string",
        "motivation": "string",
        "voice_description": "string",
        "relationships": [{ "with_character_id": "string", "relationship": "string" }],

### Agent 5：VisualGen（视觉生成）

**文件**：`app/agents/visual_gen.py`

**输入**：`{ "scenes": [...], "characters": [...], "genre": "string", "mood_tags": [...] }`

**输出**：
```json
{
  "agent_name": "VisualGen",
  "timestamp": "ISO8601",
  "data": {
    "visual_assets": [
      {
        "asset_id": "string（asset_01...）",
        "scene_id": "string",
        "character_id": "string（可选）",
        "asset_type": "string（image/video_clip/background）",
        "prompt": "string（英文 AI 生成 prompt）",
        "negative_prompt": "string",
        "generation_params": { "model": "string", "resolution": "string", "seed": "number" },
        "output_path": "string",
        "output_url": "string",
        "status": "string（pending/generating/done/failed）"
      }
    ]
  }
}
```

**技术说明**：
- 并行生成：同一 scene 的多个镜头用 `asyncio.gather` 并发生成
- 调用 `VideoGeneratorService.generate_image(prompt)` 或 `generate_video_clip(prompt)`
- 失败自动重试 2 次

### Agent 6：VideoAssembly（视频剪辑合成）

**文件**：`app/agents/video_assembly.py`

**输入**：`{ "visual_assets": [...], "scenes": [...], "characters": [...], "mood_tags": [...] }`

**输出**：
```json
{
  "agent_name": "VideoAssembly",
  "timestamp": "ISO8601",
  "data": {
    "final_video_path": "string",
    "final_video_url": "string",
    "duration_sec": "number",
    "resolution": "string（1920x1080）",
    "codec": "string（h.264）",
    "fps": "number（24）",
    "assembly_edl": "string（EDL 格式剪辑决策列表）",
    "scenes_in_order": ["string"],
    "file_size_mb": "number"
  }
}
```

**技术说明**：
- 调用 `VideoAssemblyService.assemble(assets, scenes)`
- 内部调用 FFmpeg 或视频生成 API 按 scene 顺序拼接

---

### Agent 7：QCReview（质检复审，异步）

**文件**：`app/agents/qc_review.py`

**输入**：`{ "final_video_path": "string", "final_video_url": "string", "scenes": [...], "characters": [...], "story_summary": "string" }`

**输出**：
```json
{
  "agent_name": "QCReview",
  "timestamp": "ISO8601",
  "data": {
    "quality_score": "number（0-100）",
    "issues": [
      {
        "severity": "string（critical/major/minor）",
        "type": "string（audio/video/consistency/pace）",
        "description": "string",
        "affected_scene_id": "string（可选）",
        "suggestion": "string"
      }
    ],
    "recommendations": ["string"],
    "passed": "boolean"
  }
}
```

**说明**：异步执行，不阻塞主流程，结果通过 WebSocket `qc_complete` 推送

## 完整数据流

```
story_text
    ↓
[Agent1: ScriptAnalysis] → memory.add → story_summary, genre, tone, themes...
    ↓
[Agent2: PlotDeconstruct] → memory.add → act_structure, key_plot_points, conflict
    ↓
[Agent3: ScenePlanning]  → memory.add → scenes[], total_duration
    ↓              ↕（并行，asyncio.gather）
[Agent4: CharacterDesign] → memory.add → characters[], narrator
    ↓
[Agent5: VisualGen]      → memory.add → visual_assets[], character_sheets
    ↓
[Agent6: VideoAssembly]  → memory.add → final_video_path, duration
    ↓（异步，不阻塞）
[Agent7: QCReview]       → memory.add → quality_score, issues[]
```

## 错误处理规范

| 错误场景 | 处理策略 |
|---------|---------|
| 单个 Agent 执行失败 | 重试 2 次 → story status = error → WebSocket 推送 error |
| 人工干预超时（30min） | 自动确认，继续下一 Agent |
| 视频生成 API 不可用 | 降级到 FFmpeg 方案（需备选素材） |
| 向量数据库连接失败 | 降级到纯内存模式（无长期记忆，pipeline 继续） |
| LLM API 限流 | 指数退避重试（最多 3 次） |
| Session 并发超限 | 排队等待（Semaphore 控制） |
