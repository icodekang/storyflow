## 快速启动指南（coder 参考）

### 前端启动
```bash
cd storyflow/frontend
npm install
npm run dev
# 访问 http://localhost:5173
```

### 后端启动
```bash
cd storyflow/backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
# 访问 http://localhost:8000/docs
```

### 联调
- 前端 `vite.config.ts` 配置代理：`/api` → `http://localhost:8000`
- WebSocket：`ws://localhost:8000/ws/stories/{story_id}`

## 关键配置项（backend/app/core/config.py）

```python
# 向量数据库
VECTOR_DB_TYPE: Literal["chroma", "milvus", "qdrant"] = "chroma"
VECTOR_DB_PATH: str = "./data/vector_db"

# 并发控制
MAX_CONCURRENT_STORIES: int = 3

# 人工干预超时（秒）
INTERVENTION_TIMEOUT_SEC: int = 1800  # 30分钟

# 视频存储
STORAGE_TYPE: Literal["local", "oss"] = "local"
STORAGE_LOCAL_PATH: str = "./data/storage"

# 视频生成 API（可切换）
VIDEO_GENERATOR_TYPE: Literal["local_ffmpeg", "runway", "pika"] = "local_ffmpeg"
VIDEO_GENERATOR_API_KEY: str = ""
```
